# 📊 Sales Performance Dashboard (Excel)

## 📌 Objective
Analyze sales data to identify trends, top-performing categories, and customer insights.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

Built an interactive dashboard to support business decision-making.
